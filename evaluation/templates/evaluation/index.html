Welcome to the evaluation of the Semantic Tags for Open Data Portals - STODaP system. This tool was built to facilitate the access to open datasets and thus contribute to the development of open data.

Open Data is a worldwide movement towards turning public data more accessible and reusable. Hundreds of national and local governments around the globe are currently investing in publishing their datasets of public interest in open formats, i.e.:

  • Using open and machine readable formats (CSV, RDF, XML) instead of PDF;
  • Licensing data so that it can be freely reused.

Unfortunately, very little effort has been taken in enhancing metadata related to open data. This is one the aspects that still hinders open data development.

The STODaP server aims to tackle this issue. Metadata of around 90 Open Data Portals were collected in order to build the STODaP database, which is composed by 420,000 datasets and 290,000 tags. These data is semantically grouped in 3,000 Semantic Tags, organized in 74 Semantic Groups. Semantic Tags and Groups are linked to the Linked Open Data cloud via the GEMET thesaurus.

In this evaluation, we want to measure if STODaP is really an efficient tool for finding data. Can you help us?
